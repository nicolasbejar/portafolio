## ğŸ“¦ Getting Started

### Prerequisites

- A modern web browser
- Git (for version control)
- GitHub account (for hosting)

### Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/nicolasbejar/stanford-portfolio.git
   cd stanford-portfolio
   ```

2. **Open locally**:
   - Simply open `index.html` in your browser
   - Or use a local server:
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js (if you have live-server installed)
     npx live-server
     ```

3. **View the website**:
   - Open `http://localhost:8000` in your browser

## ğŸŒ Deployment Options

### GitHub Pages (Recommended)

1. **Create a new repository** on GitHub
2. **Push your code**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Stanford portfolio website"
   git branch -M main
   git remote add origin https://github.com/yourusername/stanford-portfolio.git
   git push -u origin main
   ```

3. **Enable GitHub Pages**:
   - Go to repository Settings
   - Scroll to "Pages" section
   - Source: Deploy from a branch
   - Branch: `main` / `root`
   - Click "Save"

4. **Access your site**:
   - Your site will be available at: `https://yourusername.github.io/stanford-portfolio`
